SubscriptionType
================

:包路径: org.yuanrong.datasystem.stream

配置订阅模式的枚举类。

枚举值
------

.. list-table::
   :widths: 40 60
   :header-rows: 1

   * - 定义
     - 说明
   * - ``SubscriptionType.STREAM``
     - 流模式订阅。
   * - ``SubscriptionType.ROUND_ROBIN``
     - 轮询模式订阅。
   * - ``SubscriptionType.KEY_PARTITIONS``
     - 键分区模式订阅。
