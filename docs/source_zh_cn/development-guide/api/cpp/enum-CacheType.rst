CacheType
========================================

.. cpp:class:: CacheType

    :header-file: #include <datasystem/object/object_enum.h>
    :namespace: datasystem

    `CacheType` 类定义对象的缓存介质。

    目前，支持以下 `CacheType`：

    ===================================  ==================================================================
    定义                                 说明
    ===================================  ==================================================================
    `CacheType::MEMORY`                   内存介质，数据将缓存在共享内存上。
    `CacheType::DISK`                     磁盘介质，数据将缓存在磁盘上。
    ===================================  ==================================================================
