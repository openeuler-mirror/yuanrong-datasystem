StreamMode
========================================

.. cpp:class:: StreamMode

    :header-file: #include <datasystem/stream/stream_config.h>
    :namespace: datasystem

    `CacheType` 类定义流缓存的模式配置。

    目前，支持以下 `StreamMode`：

    ===================================  ==================================================================
    定义                                 说明
    ===================================  ==================================================================
    `StreamMode::MPMC`                   多生产者多消费者
    `StreamMode::MPSC`                   多生产者单消费者
    `StreamMode::SPSC`                   单生产者单消费者
    ===================================  ==================================================================
