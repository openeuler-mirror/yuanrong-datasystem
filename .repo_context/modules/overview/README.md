# Overview Context

`overview/` contains repository-wide orientation documents.

Use this directory first when you need the broad structure of the repo before diving into a domain-specific area.

Current docs:

- `engineering-principles.md`: repository-level infrastructure engineering standards for performance, concurrency,
  persistence, recovery, testing, review, and AI-agent self-verification.
- `repository-overview.md`: coarse map of the major source, SDK, CLI, docs, and test areas.
- `repository-skills.md`: repository-local Codex skill inventory and natural-language trigger routing rules.
