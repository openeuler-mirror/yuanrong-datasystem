# Infra Feature Playbooks

`playbooks/features/infra/` contains low-risk implementation workflows for shared infrastructure features.

Use this directory when a feature touches cross-cutting infrastructure such as logging, metrics, transport, storage backends, shared memory, or other common runtime plumbing.
